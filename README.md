1. Download the folders and files into a new project folder
2. Open the project folder into an IDE (VS Code recommended)
3. Run the program in the IDE (press F5 if using VS Code)
4. Executing the program opens a web page in Edge with the application
5. In the application, go to the A* Visualizer tab
6. Click on the grid cells to add/remove walls
7. Click on Run A* to visualize the shortest path (shown by blue cells) from the start (green cell) to the end (red cell)
